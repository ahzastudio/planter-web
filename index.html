Cara Install Toolbox

Ikuti langkah mudah berikut untuk menambahkan Plantation Tools ke ArcGIS Pro Anda

1

Download & Extract

Pilih versi yang sesuai (ArcGIS Pro atau ArcMap). Download file ZIP dari tombol di atas. Extract file ZIP ke folder pilihan Anda (misal: C:\MyToolboxes).

2

Buka ArcGIS Pro

Buka ArcGIS Pro → Klik tab View → Klik Catalog Pane. Di Catalog Pane, klik kanan pada Toolboxes → Pilih Add Toolbox.

3

Pilih File .pyt

Browse ke folder hasil extract → Pilih file PlantationTools.pyt (Pro) atau PlantationTools_ArcMap.pyt → Klik OK. Toolbox siap digunakan! 🎉

Catatan Penting

  • • Koneksi Internet: Diperlukan saat pertama kali menggunakan tool untuk aktivasi trial otomatis.
  • • Model AI (Opsional): Jika menggunakan fitur AI Palm Detection, download model ONNX dan letakkan di folder yang sama dengan file .pyt
  • • Trial Otomatis: Saat pertama kali menjalankan tool, Anda otomatis mendapat trial 14 hari dengan akses penuh fitur PRO.
